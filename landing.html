How it works

The front end is written in angularJS that consumes rest Json services provided by a java web app that scrapes Scaruffi's website and insert that data into a database. The source code for both can be found on my github.

Given that Scaruffi's knowledge base predates the internet it is not very consistent in formatting and that reflects here. If you find a dead link or an incomplete page you can use google to find the original page or follow the link to the original page by clicking the band or film name at the top.
